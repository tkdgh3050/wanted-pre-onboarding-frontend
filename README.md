## 🐻 프로젝트의 실행 방법
### `1. npm install`
프로젝트에 필요한 패키지 설치
### `2. npm start`
[http://localhost:3000](http://localhost:3000) 에 개발서버가 실행되며 접속 시 사이트 확인 가능
